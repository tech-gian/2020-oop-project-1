### Documentation

    Σκοπός της άσκησης αυτής είναι η προσομοίωση της πρωινής άφιξης των μαθητών σε
    σχολείο. Για να γίνει αυτό, αρχικά, έχω ορίσει τις class μου στο αρχείο
    classes.h, στο οποίο υπάρχουν όλα όσα χρειάζεται κάποιος για να καταλάβει την δομή
    των χώρων του σχολείου, καθώς και συναρτήσεις για την λειτουργικότητα του κάθε χώρου
    (συνοδευόμενες με τα απαραίτητα σχόλια).

    Στο αρχείο classes.cc υπάρχουν οι υλοποιήσεις για όλες τις συναρτήσεις που αναφέρονται
    στο .h αρχείο (εκτός από εκείνες που έχουν υλοποιηθεί στο .h, ώστε να λειτουργούν ως
    inline συναρτήσεις). Σε κάθε χώρο υπάρχει 1 συνάρτηση constructor και 1 destructor, οι
    οποίες καλούνται κάθε φορά που δημιουργείται/καταστρέφεται η αντίστοιχη class. Επίσης σε
    κάθε class υπάρχει κι η αντίστοιχη συνάρτηση print, που εκτυπώνει τα απαραίτητα
    στοιχεία/μηνύματα.
    
    Υπάρχουν οι class για student και teacher, που αποθηκεύουν τα στοιχεία του κάθε μαθητή και
    αντίστοιχα του κάθε καθηγητή. Συνεχίζοντας, η class school, διατηρεί όλους τους χώρους που
    υπάρχουν σε ένα σχολείο, (εκτός από βοηθητικές συναρτήσεις) έχει 2 (overloading)
    συναρτήσεις enter και μια place για τοποθέτηση δασκάλου. Η μία από αυτές δέχεται έναν μαθητή
    και τον εισάγει στο σχολείο (ουσιαστικά στην αυλή) και η άλλη δέχεται ένα σύνολο μαθητών
    για να εισαχθούν στο σχολείο.

    Αντίστοιχα, στην class yard υπάρχουν οι enter, exit που η πρώτη εισάγει μαθητή στο προαύλιο (αν
    χωράει) και η δεύτερη αφαιρεί τον τελευταίο μαθητή που εισήχθει (αν δεν είναι άδειος ο χώρος).
    Στην class stairs υπάρχουν οι ίδιες συναρτήσεις εισόδου και εξόδου και επιτελούν παρόμοια διαδικασία,
    απλά αυτήν την φορά για το κλιμακοστάσιο.

    Η class flo αντιπροσωπεύει έναν όροφο, που περιέχει έναν διάδρομο κι κάποιες τάξεις. Πέρα από
    κάποιες βοηθητικές συναρτήσεις που υπάρχουν, υπάρχει ακόμα η enter που εισάγει 1 μαθητή (αν χωράει)
    στον όροφο (ουσιαστικά στον διάδρομο) και η exit που αφαιρεί τον τελευταίο μαθητή που εισήχθει (αν δεν
    είναι άδειος ο χώρος). Η class corridor έχει κι αυτή μια enter και μια exit, που επιτελούν παρόμοια
    διαδικασία με τις υπόλοιπες enter/exit και μια place που εισάγει (ακαριαία) έναν καθηγητή στην τάξη του.

    Τέλος, class classroom περιέχει κι αυτή μια enter, που εισάγει 1 μαθητή στην τάξη (αν χωράει και
    δεν έχει μπει ο καθητής ακόμα).


    Στην main (στο αρχείο main.cc) δημιουργούνται μαθητές και καθηγητές (στην συγκεκριμένη περίπτωση από
    το αρχείο input.txt) και δημιουργείται και το σχολικό κτήριο. Στην συνέχεια, επαναλαμβάνεται η ακόλουθη
    διαδικασία:
        - Επιλέγεται τυχαία αν θα μπει μαθητής ή καθηγητής
            - Αν μπει μαθητής, ακολουθεί την μέγιστη δυνατή πορεία που μπορεί να κάνει
            - Αν μπει καθηγητής (και δεν βρίσκεται ήδη στο σχολείο) τοποθετείται στην τάξη του
        - Ξαναγίνεται η ίδια διαδικασία έως ότου:
            - Έχουν μπει όλοι οι μαθητές στο σχολείο Ή
            - Δεν χωράει να μπει άλλος μαθητής
    Ύστερα, εκτυπώνεται το σχολείο και αποδεσμεύεται όλος ο χώρος που έχει χρησιμοποιηθεί.


    Επίσης, υπάρχει κι ένα απλό Makefile, στο οποίο κάποιος μπορεί να αλλάξει τα arguments που δέχεται η main
    από την μεταβλητή ARGS. Για να γίνουν compile τα αρχεία πρέπει να τρέξει η εντολή make. Για να τρέξει το
    πρόγραμμα, η εντολή make run. (Οι επιλογές clean και valgrind είναι αντίστοιχα για να διαγραφούν τα έξτρα
    αρχεία που δημιουργήθηκαν και για να τρέξει το πρόγραμμα με valgrind, ώστε να φανούν τα leaks, αν υπάρχουν).
    Για την εντολή μεταγλώττισης πιο αναλυτικά, φαίνεται στο Makefile.

    [Η εντολή μεταγλώττισης: g++ -o main main.cc classes.cc
    και η εντολή για να τρέξει: ./main 2 5 5 5 < input.txt]

    [Το πρόγραμμα τρέχει σωστά και χωρίς leaks σε linux (εκτός από ένα bug που υπάρχει στην έκδοση του valgrind
    και εμφανίζει 72,704 bytes still reachable) της σχολής]
